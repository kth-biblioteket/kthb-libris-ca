Help

Cloud App which connects records in Alma to Libris Holdings.

This app is still in BETA-version.

How to use

Search for physical titles. For instance “mathematics”. The app will go through the list of Titles and get the Libris Holdings and show them in the app(if they exist in the external system). The Title is also a link to the main bib in “Libris” and then each holding has a link to the holding in “Libris”

Configuration

In the configuration page the Sigels and URL to “Libris” are defined.

Example: Sigels for KTH: T, Te and Tct.

Link to libris: https://libris.kb.se (https://libris-qa.kb.se for Libris sandbox)

The proxy field is not used at the moment (this might be used in a future version.)
